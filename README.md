# docker-flocker-pki

A data container that generates and stores the PKI for a Flocker cluster.

## Usage

### Build

    $ docker build -t flaccid/flocker-pki .

### Run

    $ docker run -it flaccid/flocker-pki

### Publish

    $ docker push flaccid/flocker-pki

Drops to a shell only.
